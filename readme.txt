A professional candidate application management plugin for WordPress. Collect applications, resumes, and documents directly on your website with a customizable form and admin dashboard.

== Description ==

**Candidate Manager** is a WordPress plugin that allows website owners to collect and manage job applications directly from their website. It provides:

- A professional application form with resume and license upload support
- An admin dashboard to view, filter, and manage candidates
- Anti-spam protection (math puzzle)
- Shortcode `[candidate_form]` to embed the form anywhere
- Database storage for secure candidate management

Perfect for businesses, recruiters, and HR professionals who want a simple yet powerful way to collect job applications.

== Features ==

* Professional, styled candidate application form
* Resume and driver license uploads
* Customizable available positions
* Secure anti-spam puzzle
* Admin dashboard for managing candidates
* Export functionality (CSV/Excel)
* Shortcode `[candidate_form]` for easy embedding
* Works with any WordPress theme

== Installation ==

1. Download the plugin ZIP or clone it from GitHub.  
2. Upload the plugin folder to `/wp-content/plugins/`.  
3. Activate the plugin from the WordPress **Plugins** menu.  
4. Use the shortcode `[candidate_form]` to display the application form.  

== Frequently Asked Questions ==

= Can I customize the available positions? =  
Yes, you can easily modify the list of positions in the plugin settings or code.  

= Where are resumes stored? =  
Uploaded files are stored in your WordPress `uploads` directory and linked in the database for admin review.  

= Does it protect against spam? =  
Yes, the form includes a math puzzle for anti-spam protection. Optionally, you can add Google reCAPTCHA.  

= Can I export candidate applications? =  
Yes, the admin dashboard includes export options for CSV or Excel.
